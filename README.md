# Практическая работа: Развертывание Moodle

## Как запустить проект

1. Откройте терминал в этой папке
2. Выполните команду: `docker-compose up -d`
3. Подождите 1-2 минуты пока все загрузится
4. Откройте браузер и перейдите по адресу: http://localhost:8080
5. Логин: admin
6. Пароль: AdminPass123!

## Структура проекта
- `content/` - учебные материалы
- `scripts/` - скрипты бэкапа
- `docs/` - документация и скриншоты
- `videos/` - демонстрационные видео
- `load_test/` - тестирование производительности
