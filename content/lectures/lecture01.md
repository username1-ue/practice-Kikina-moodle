# Лекция 1 — Что такое файлы и файловые системы

**Тема:** Основы работы с файлами
**Цель урока:** Понять, что такое файлы, типы файлов, файловые системы
**Время:** 20 мин.
**Ключевые понятия:** файл, папка, файловая система, расширение, путь к файлу

## 1) Введение
Файл — это поименованная область данных на носителе информации. Все данные на компьютере хранятся в файлах: документы, фотографии, программы.

## 2) Основная часть
- **Имя файла** состоит из имени и расширения (например, document.docx)
- **Расширение** показывает тип файла: .txt (текст), .jpg (картинка), .mp3 (музыка)
- **Папка (директория)** — контейнер для файлов
- **Файловая система** — способ организации данных на диске (NTFS, FAT32, ext4)
- **Путь к файлу** — адрес, по которому можно найти файл (C:\Users\Имя\file.txt)

## 3) Пример
На Windows: C:\Users\Student\Documents\hw.txt
На Linux: /home/student/documents/hw.txt
