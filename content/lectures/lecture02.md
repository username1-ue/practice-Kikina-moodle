# Лекция 2 — Работа с файлами в командной строке

**Тема:** Основные команды для работы с файлами
**Цель урока:** Научиться создавать, копировать, перемещать и удалять файлы
**Время:** 20 мин.
**Ключевые понятия:** команда, терминал, копирование, перемещение, удаление

## 1) Введение
Командная строка — мощный инструмент для работы с файлами. Она быстрее графического интерфейса для многих операций.

## 2) Основная часть

**Windows (cmd):**
- `dir` — посмотреть содержимое папки
- `cd папка` — перейти в папку
- `copy file1.txt file2.txt` — копировать файл
- `move file.txt папка` — переместить файл
- `del file.txt` — удалить файл

**Linux/Mac:**
- `ls` — посмотреть содержимое
- `cp file1.txt file2.txt` — копировать
- `mv file.txt папка/` — переместить
- `rm file.txt` — удалить

## 3) Пример
Создать папку project, перейти в неё, создать файл readme.txt:
mkdir project
cd project
echo "Hello" > readme.txt
