# Лекция 3 — Резервное копирование файлов

**Тема:** Создание резервных копий
**Цель урока:** Научиться делать бэкапы важных файлов
**Время:** 20 мин.
**Ключевые понятия:** бэкап, копия, восстановление, версионность

## 1) Введение
Резервное копирование (backup) — создание копии данных на случай их потери.

## 2) Основная часть
**Правила бэкапов:**
- Правило 3-2-1: 3 копии, 2 разных носителя, 1 копия вне офиса
- Делать бэкапы регулярно
- Проверять, что бэкапы работают

**Способы бэкапа:**
1. Простое копирование (cp, copy)
2. Архивирование (zip, tar)
3. Синхронизация (rsync, robocopy)
4. Системы контроля версий (Git)

## 3) Пример
Простой бэкап на Linux:
```bash
cp -r важная_папка /backup/важная_папка_$(date +%Y%m%d)
