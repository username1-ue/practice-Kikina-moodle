# Лекция 4 — Git для резервного копирования

**Тема:** Использование Git как системы бэкапов
**Цель урока:** Научиться использовать Git для хранения версий файлов
**Время:** 20 мин.
**Ключевые понятия:** Git, репозиторий, коммит, ветка

## 1) Введение
Git — система контроля версий. Позволяет хранить историю изменений файлов.

## 2) Основная часть
**Основные команды Git:**
- `git init` — создать репозиторий
- `git add файл` — добавить файл в отслеживаемые
- `git commit -m "описание"` — сохранить изменения
- `git log` — посмотреть историю
- `git checkout` — вернуться к старой версии

## 3) Пример
```bash
mkdir myproject
cd myproject
git init
echo "Version 1" > file.txt
git add file.txt
git commit -m "First version"
echo "Version 2" > file.txt
git commit -am "Second version"
git log
